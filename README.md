# Class Site Template

이 템플릿은 GitHub Pages에 바로 올릴 수 있는 가장 간단한 구조입니다.

## 구조
```
.
├─ index.html
├─ README.md
├─ images/
│   └─ icon.png
└─ exercises/
    └─ song.html
```

## 사용법
1. 이 폴더 내용을 본인 저장소 루트에 복사합니다.
2. `index.html`을 수정하여 내용을 채웁니다.
3. 커밋(Commit) → 푸시(Push)합니다.
4. 저장소의 **Settings → Pages**에서 배포 브랜치를 `main`/`/ (root)`로 설정합니다.
